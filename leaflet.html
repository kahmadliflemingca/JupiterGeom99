Leaflet

Leaflet is an open-source JavaScript library used for creating interactive maps on the web. The Leaflet JavaScript library contains approximately 42 KB of JavaScript and is customizable and extendable, with various plugins. One of these plugins is the Esri Leaflet plugin, a lightweight and open-source tool that enables access to ArcGIS services.

+

  • Pros

  • Open source and Free
  • Easy to Use
  • Highly customizable and extendable, with various plugins
  • Allows for easy display of GeoJSON on a single map
  • Offers custom-designed widgets such as layer control, Basemap gallery, and Measure tool, and etc.

  • Cons

  • Does not offer Security
  • If data is sensitive requires Server to host the data
  • Performance: It may not perform well when dealing with large dataset
  • Dependency on third-party libraries
  • Lack of support: Because Esri Leaflet is a community-supported library, it may not receive the same level of support or updates as other Esri products or services

What Didn't Work

If data is sensitive and shouldn't be publicly available, it requires a secure storage environment that requires authentication for access. While Leaflet is a great option for displaying publicly available data, it was not suitable for our client's sensitive data. After researching various solutions, we concluded that setting up a server (such as Geoserver) and hosting it on cloud services like Google Console or AWS was the best option to host the data. However, having a dedicated server for the client was not practical due to budget constraints. As a result, we needed a cost-effective alternative to securely host the sensitive data.

What Did Work

Working with GeoJSON files

  • Convert shapefile to GeoJSON in arcgis pro using Features to Json tool
  • Evaluate attributes of the shapefile, define which attributes should be included to the MapViewer
  • Host Geojson files in Github.

    • Steps to develop the Map Viewer

  • Include a Leaflet library from a content delivery network (CDN)
  • Add the Esri plugin to enable the use of Esri basemaps and services in your map.
  • Add the measure plugin to allow users to measure distances and areas on the map.
  • Create a map object and set its initial view, specifying the center point and zoom level.
  • Define several Esri basemaps using the L.esri.Vector.vectorBasemapLayer function and create an object called vectortiles to hold them.
  • Add the basemaps to a layer control object using the L.control.layers function, so that users can switch between them easily
  • To add GeoJSON layers to the map, use an XMLHttpRequest to fetch the GeoJSON data from Github
  • Create a new Leaflet GeoJSON layer object and add it to the map, using the onEachFeature option to define a popup for each feature in the layer
